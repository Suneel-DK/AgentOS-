# AgentOS

An open-source operating system built for AI — flash it, boot it, and your AI stack is ready to run. No setup, no terminal, no complexity.

## Why AgentOS?
Running local AI today is painful — GPU drivers, dependencies, configuration. AgentOS eliminates all of that.

## Vision
- Auto-detects your hardware (NVIDIA, AMD, CPU)
- Comes pre-installed with Ollama, Open WebUI, and model management
- Simple browser dashboard — no terminal needed
- Built for AI agents to run locally, privately, and fast
